# Fraud Detection for E-commerce and Bank Transactions

## Overview
This project focuses on improving fraud detection for e-commerce and bank credit transactions using machine learning. The goal is to accurately identify fraudulent activities while balancing security and user experience.

Two datasets are used:
- E-commerce transaction data with behavioral and geolocation features
- Bank credit card transactions with anonymized PCA features

The project addresses severe class imbalance and emphasizes explainable AI.

---

## Business Problem
False positives frustrate customers, while false negatives cause financial loss.  
This project builds fraud detection models optimized for imbalanced data and business cost trade-offs.
